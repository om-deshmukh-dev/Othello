## Installation Instructions

### Prerequisites
- OCaml (version 4.14 or higher)
- OPAM (OCaml Package Manager)
- Dune build system

### Setup

1. Clone the repository:
   ```
   git clone <repository-url>
   cd <repository-directory>
   ```

2. Install dependencies (if any):
   ```
   opam install . --deps-only
   ```

3. Build the project:
   ```
   dune build
   ```

### Running the Game

4. Start the game:
   ```
   make othello
   ```

5. Choose game mode:
   - For singleplayer: type 's [difficulty]' where difficulty is easy, medium, hard, or godmode
     - Example: `s medium`
   - For multiplayer: type 'm'
   - Type 'q' to quit or 'h' for help

6. If playing against computer, choose your color:
   - 'b' for black
   - 'w' for white

7. Enter moves as (letter, number), e.g., 'f 5'
   - Valid rows: 1-8
   - Valid columns: A-H
   - The board displays black pieces (⚫) and white pieces (⚪)

8. Additional commands during gameplay:
   - 'history' - Enter history mode to revert to a previous game state
   - 'q' - Quit the game
   - 'h' - Show available commands

9. When finished, clean the build:
   ```
   dune clean
   ```
